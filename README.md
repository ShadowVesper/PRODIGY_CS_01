# PRODIGY_CS_01

## Caesar Cipher Tool

This project was created as part of my Prodigy InfoTech Cybersecurity Internship.

### Features
- Encrypts messages using the Caesar Cipher.
- Decrypts encrypted messages.
- Accepts a custom shift value.
- Preserves spaces and punctuation.
- Validates user input (Encrypt or Decrypt).

### Technologies Used
- Python

### Example

Input:
Message: PRODIGY_CS_01
Shift: 3

Encrypted:
SURGLJB_FV_01

Decrypted:
PRODIGY_CS_01
